# Amd


La carpeta AMD corresonde a un directorio dedicado a la definición de módulos asincrónicos en javascript. Dentro de esta carpeta se encuentran archivos que corresponden a la definición previa de un mánager, módulos en javascript que pueden ser utilizados en más de un lugar y bibliotecas adicionales en javascript.



## Estructura

En la carpeta *amd* se maneja la siguiente estructura:


* amd:
  - componentes: *.component.js

Los archivos en este directorio se clasifican en dos carpetas: *src* y *build*. 
- Los archivos en la carpeta *src* son desarrollados en el módulo en javascript
- Los archivos en la carpeta *build* son los archivos javascript *minificados*, estos archivos los lee el campus
